# 这是一个vuex + nuxt的项目

------
本项目仅供学习使用，如需转载或下载请注明出处。
项目技术栈：

> * vue & vuex
> * nuxt
> * sass
> * pug模板（jade）
> * nodejs
> * mysql

------

## 目录解析

> assets  静态页面中使用到的资源
> build  nuxt build后的js目录
> components  静态页面通用组件库
> layouts  静态页面布局组件
> page  静态页面组件
> plugins  插件
> server  nodejs服务端
> static  静态页面样式
> store  vuex store
